Exemplos de align-items em CSS

align-items: flex-start

Seção flex-start: Os itens são alinhados no início do contêiner (no topo).

align-items: flex-end

Seção flex-end: Os itens são alinhados no final do contêiner (na parte inferior).

align-items: center

Seção center: Os itens são alinhados no centro do contêiner.

align-items: baseline

Seção baseline: Os itens são alinhados de acordo com suas linhas de base de texto. Um dos itens tem uma altura diferente para destacar isso.

align-items: stretch

Seção stretch: Os itens são esticados para preencher a altura total do contêiner (se não houver altura definida).
